设置免密SSH拷贝备份到远程服务器

设置远程无密码拷贝

数据库服务器上执行

会在用户目录~/.ssh/产生两个文件，id_rsa，id_rsa.pub

将id_rsa.pub文件拷贝到远程主机的root用户主目录下的.ssh目录下,并且改名为authorized_keys

这样数据库服务器上使用scp命令复制文件到远程主机上将不提示输入密码了，直接自动复制了
